# Himalayausa_clone
Himalaya USA is an E-Commerce Website which helps users to buy Cosmetics products and Beauty products online at best prices .

## Tech Stack Used : -

- #### Languages
  - `HTML`
  - `CSS`
  - `JavaScript ES6`
  - `React`
  - `Redux`
  - `Chakra-UI`
  - `Heroku`

#### CSS frameworks
  - `BootStrap`
